# Introducion

In this section we will introduce example of each available tool of Geovisto with working example.  

Currently available tools maps in Geovisto are:

-   **Sidebar tool**
-   **Info tool**[^1]
-   **Selection tool**[^2]
-   **Legend tool**[^3]
-   **Themes tool**[^4]
-   **Timeline tool**[^5]
-   **Filters tool**[^6]
-   **Geo-Downloader tool**[^7]
-   **Hierarchy tool**[^8]
