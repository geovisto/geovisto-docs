A spatial heat map displays the magnitude of a spatial phenomena as color, usually cast over a map.[^1]<br/>
Example below contains data about **Maximum temperature** over cities and places in Australia.

## Example

<Heat />

### Configuration

Configuration used in example above:

```json
{
    "type": "geovisto-tool-layer-heat",
    "id": "geovisto-tool-layer-heat",
    "enabled": true,
    "layerName": "Heat layer",
    "data": {
        "latitude": "lat",
        "longitude": "long",
        "radius": 20,
        "intensity": "temperature",
        "gradient": "Default",
        "blur": 23,
        "zoom": "Normal"
    }
}
```

## Dimensions

Dimensions used in Heat Layer:

-   `latitude` - latitude
-   `longitude` - longitude
-   `radius` - radius of each point
-   `intensity` - intesity of each point
-   `gradient` - color gradient
-   `blur` - amount of blur
-   `zoom` - zoom level where the points reach maximum intensity
