The Legend provides a more detailed and clear overview of the visualized data.

## Example

<Legend />

### Installation

```
npm i geovisto-legend
```


### Configuration

Configuration used in example above:

```json
{
    "type": "geovisto-tool-legend",
    "id": "geovisto-tool-legend",
    "enabled": true,
    "tools": [
        "geovisto-tool-layer-choropleth"
    ]
}
```
