Filtering enables the selection of objects according to one or more specified conditions.
## Example

<Filters />

### Installation

```
npm i geovisto-filters
```


### Configuration

Configuration used in example above:

```json
{
    "type": "geovisto-tool-filters",
    "id": "geovisto-tool-filters",
    "enabled": true,
    "filterRules": [
        {
            "domain": "Year",
            "operation": "==",
            "pattern": "2019"
        },
    ]
}
```
