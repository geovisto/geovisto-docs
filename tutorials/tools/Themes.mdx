Using the themes tool, the user can set a favorite theme.

## Example

<Themes />

### Installation

```
npm i geovisto-themes
```

### Configuration

Configuration used in example above:

```json
{
    "type": "geovisto-tool-themes",
    "id": "geovisto-tool-themes",
    "enabled": true,
    "theme": "light1"
}
```
