Tool provides ability to manage displayed geographic objects (point, polygon) according to zoom level of map. Data value of geographical object can be obtained from its hierarchicly lowest objects.

## Example

<Hierarchy />

### Installation

```
npm i geovisto-hierarchy
```

### Configuration

Configuration used in example above:

```json
    {
        "type": "geovisto-geo-downloader",
        "id": "geovisto-geo-downloader",
        "enabled": true
    }
```
