The sidebar is the basic Geovisto library tool that can be used to edit visualized data.

<Sidebar />

### Installation

```
npm i geovisto-sidebar
```

### Configuration

Configuration used in example above:

```json
{
      "type": "geovisto-tool-sidebar",
      "id": "geovisto-tool-sidebar",
      "enabled": true,
      "tabs": []
}
```
