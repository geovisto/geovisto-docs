Provides user interface for downloading points or polygons from OverpassAPI (OSM) of selected country. 

## Example

<GeoDownloader />

### Installation

```
npm i geovisto-geo-downloader
```


### Configuration

Configuration used in example above:

```json
{
    "type": "geovisto-geo-downloader",
    "id": "geovisto-geo-downloader",
    "enabled": true
}
```
